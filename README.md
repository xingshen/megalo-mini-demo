# hello-megalo  

## 安装megalo  
[参考官网](https://megalojs.org/#/quickstart)

## 编译  
```bash
npm install
```

## 运行

微信小程序：

```bash
npm run dev:wechat
```

支付宝小程序：

```bash
npm run dev:alipay
```

百度智能小程序：

```bash
npm run dev:swan
```
生成的dist-xx目录就是小程序的工程包，可以使用相应的开发者工具导入此包查看效果
